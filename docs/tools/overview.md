# Creator & developer tools catalog

Alongside the AI content studio, ContentSynergy AI ships a catalog of standalone utilities that run entirely in your browser. They share the same account and theme as the rest of the app, and each one saves its results to a matching "My …" history page so you can find past work. Everything here is client-side and free — no uploads to a paid service, no usage cap.

Every tool has two halves: a **create** page where you produce something, and a **My …** page that lists what you've produced. The lists are searchable and sortable.

## Images & media

Produce and transform visual assets without leaving the app.

- **Image Generator** — styled image compositions across 12 styles and 12 moods at 6 aspect ratios.
- **Background Remover** — strip backgrounds from images client-side.
- **Image Compressor** — reduce file size while controlling quality.
- **Image Converter** — convert between common raster formats.
- **Image Cropper / Resizer** — crop to a region or resize to exact dimensions.
- **Watermark** — overlay text or image watermarks.
- **GIF Creator** — assemble frames into an animated GIF.
- **Screenshot Tool** — capture and frame screenshots.
- **SVG Optimizer** — shrink SVG markup losslessly.
- **OG Image / Preview** — design and preview social share images.
- **Placeholder Generator** — generate placeholder images for mockups.

## Color & branding

Build a consistent visual identity.

- **Color Palette** — generate and save harmonious palettes.
- **Color Picker** — sample and store colors.
- **Gradient Generator** — design CSS gradients.
- **Shadow Generator** — craft layered CSS box-shadows.
- **Favicon Generator** — produce favicons in standard sizes.
- **Icon Generator** — generate simple icon assets.
- **Brand Kit** — collect logo, colors, and fonts into one kit.
- **Business Card** — lay out a printable business card.
- **Font Pairing** — find and preview complementary type pairings.
- **Logo Mockup** — preview a logo across mockup surfaces.

## Password & security

- **Password Generator** — produce strong passwords with configurable rules.

## How the tools fit the workflow

Because the tools live next to the content studio, a typical session flows naturally: draft a post with the text generator, run it through the readability and grammar checkers, build a [color palette](/tools/overview) and pair it with a generated image, then export an OG image for the share preview. Each result is saved to its history page so you can reuse it later.

## Frequently asked questions

**Are the tools free?** Yes, all of them, with no usage cap. They run client-side.

**Do my files get uploaded anywhere?** The tools process in the browser. Where you choose to persist a result, it's stored to your account (Firestore) or via FilesHub — never a paid third-party processor.

**Can I find my past results?** Yes. Every tool has a matching "My …" page listing prior outputs, with search and sort.
